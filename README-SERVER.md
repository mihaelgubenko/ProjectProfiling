# Инструкция по запуску сервера

## Установка зависимостей

```bash
npm install
```

## Запуск сервера

```bash
npm start
```

Сервер запустится на `http://localhost:3000`

## Настройка Telegram

1. **Создание бота:**
   - Откройте Telegram и найдите @BotFather
   - Отправьте команду `/newbot`
   - Следуйте инструкциям для создания бота
   - Скопируйте токен бота

2. **Получение Chat ID:**
   - Откройте Telegram и найдите @userinfobot
   - Отправьте команду `/start`
   - Скопируйте ваш Chat ID (число вида 123456789)

3. **Настройка config.js:**
   - Откройте файл `config.js`
   - Вставьте токен бота в `botToken`
   - Вставьте ваш Chat ID в `chatId`

## Использование

После запуска сервера, формы на сайте будут автоматически отправлять уведомления в ваш Telegram при заполнении и отправке.

## Структура проекта

- `server.js` - основной серверный файл
- `config.js` - настройки Telegram и сервера
- `package.json` - зависимости проекта
- `index.html` - главная страница
- `script.js` - клиентский JavaScript
- `styles.css` - стили сайта
