Техническое задание (ТЗ) на создание продающей страницы проектов пользователя в Cursor IDE
1. Цель проекта

Создать одностраничный продающий сайт-портфолио, представляющий цифровые проекты пользователя: Telegram-боты, автоматизации, базы данных, психологические инструменты, интерфейсы и решения для бизнеса.
Страница должна вызвать доверие, продемонстрировать компетентность и стимулировать покупку/заказ разработки.

2. Система разработки

IDE: Cursor IDE

Технологии:

HTML5

CSS3 (включая Flex/Grid)

Vanilla JavaScript (минимально, только для навигации и анимаций)

Без: React, Tailwind, внешних UI библиотек.

3. Общая структура страницы
3.1. Шапка страницы

Логотип (текстовый)

Заголовок:
“Цифровые проекты и решения под ключ”

Короткий подзаголовок:
“Автоматизация, боты, аналитика, психология — практичные инструменты для жизни и бизнеса.”

Кнопка:
“Связаться / Заказать”

4. Главные блоки страницы

Будет 10 блоков — по количеству твоих проектов.

Блок 1: Психология Души

Описание:

Комплексный инструмент для анализа снов, эмоций, внутренних состояний и глубинной психологии.
Основан на современных ИИ-технологиях, архетипах, юнгианских структурах и личных ассоциациях пользователя.
Подходит для консультаций, introspection, ведения дневника души.

Состоит из модулей:

Интерпретация сна (ИИ)

Анализ эмоционального состояния

Работа с символами

Личные архетипы

Хранилище записей

PDF-отчёты

Визуалы:

Скриншоты интерфейса (пользователь добавит- оставить место)

Пример интерпретации

Пример отчёта

Блок 2: Telegram-бот “Профайлер” (анализ анкет)
Описание:

Бот принимает анкету и формирует психологический профиль, сильные стороны, риски и рекомендации.
Используется SQLite-БД, генерация PDF-отчётов, API-интеграция GPT-3.5.

Включить:

Описание логики

Пример отчёта (без реальных данных)

Скриншоты бота

Блок 3: Telegram-бот для чеков (База данных на иврите)
Описание:

Приложение для обработки чеков:
поиск, фильтрация, редактирование, отчёты, обязательные поля, уникальные ключи.

Функции:

Структурированная БД SQLite

Привязка к полям: מספר חשבון, סכום, סיבת ביטול и др.

Полная фильтрация

Работа через GUI (DB Browser)

Материалы:

Скриншоты таблиц

Описание структуры БД

Блок 4: Бот для анализа совместимости по датам рождения
Описание:

Нумерологический движок: анализ личности, жизненного пути, совместимости.
Выводит удобный PDF или текст.

Блок 5: Бот “Отношения и психология”
Описание:

Инструмент для анализа конфликтов, эмоций, портретов личности.
Используется для консультаций.

Блок 6: Автоматизация работы с файлами

Примеры:

Генерация PDF (на основе ReportLab)

Создание DOCX, XLSX, CSV

Автоматизация отчётов

Блок 7: Проект “Эпоха 80-90-х: кем бы я был”
Описание:

Публицистический/исторический проект о роли AI-архитектора, образы профессий прошлого.

Блок 8: Проект “AI-учитель ChatGPT”

Страница с инструкциями правильных запросов, примерами работы, курсом “как задавать вопросы”.

Блок 9: Проект для трудовых споров, споров и делам по мелким искам. помошь по обращениям в разные инстанции
Описание:

Генератор структурированных обращений, жалоб, писем работодателю, помощь в обращенияхв суд
Уже есть готовые примеры.

Блок 10: Профессиональные резюме + проекты
Описание:

Список твоих проектов с кратким описанием.
Подойдёт для HR, работодателей и клиентов.

6. Требования к дизайну

Минимализм, чистый строгий стиль

Фоновый цвет — светлый

Акцентные заголовки

Небольшие анимации появления (JS)

Скриншоты расположить в карточках

Текст — крупный, легко читаемый

Визуально — как профессиональное портфолио разработчика + психолога + аналитика

7. Отзывы (фейковые, без имён)

3–4 нейтральных отзыва вида:

“Очень структурно, понятно и профессионально. Получил результат лучше, чем ожидал.”
